## artkuno

### Deskripsi Website

Artkuno adalah platform lelang online terkemuka yang mengkhususkan diri dalam barang antik dan seni rupa. Misi kami adalah menghubungkan kolektor, penjual, dan penggemar seni di seluruh dunia melalui pengalaman lelang yang aman, transparan, dan mudah diakses.

Dengan kurasi ketat dan jaringan global, kami memastikan setiap item yang ditawarkan memiliki nilai sejarah dan estetika yang tinggi.

---

## Tech Stack yang Digunakan

- **Frontend** : HTML, CSS, Tailwind CSS, JavaScript, Vite
- **Backend** : Laravel 12
- **Database** : MySQL
- **Version Control** : Git & GitHub

---

## Cara Menjalankan Website

### Clone Repository
```bash
git clone/repository
```
### Setelah Clone
```bash
cd artkunolelang/laravel
```
### buka di VSCODE
```bash
composer install
npm install
cp .env.example .env
```
### lalu sesuaikan pada .env
```bash
DB_DATABASE=artkunolelang
DB_USERNAME=root
DB_PASSWORD=
```
### generate KEY
```bash
php artisan key:generate
```
### migrasi database
```bash
php artisan migrate
php artisan storage:link
```
### jalankan server
```bash
php artisan serve
npm run dev
###jalankan di http://127.0.0.1:8000
```
